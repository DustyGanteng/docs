# Info Singkat
Halaman ini berisi commands yang dipakai di OneBlock.<br>
Nanti ada nama command, deskripsi dan contoh.

## Komponen command
Contoh: `/ob info DustyGanteng`<br>
`ob` adalah nama command<br>
`info` adalah subcommand<br>
`DustyGanteng` adalah argument

## Penanda
Jika ada tanda `[]` di command, berarti argument tidak wajib.<br>
Jika ada tanda `<>` di command, berarti argument wajib dimasukkan.
