# OneBlock
Daftar command OneBlock bisa digunakan untuk buat island, invite orang, dan masih banyak lagi.<br>

> [!TIP]
> Command **/ob** bisa diganti dengan **/is**, terserah mau pakai yang mana.

## /ob
Kegunaan: Teleport ke island atau membuat island jika belum ada<br>
Penggunaan: `/ob`<br>
Contoh dibawah:
```kominfo
/ob
```

## /ob create
Kegunaan: Membuat island<br>
Penggunaan: `/ob create`<br>
Contoh dibawah:
```kominfo
/ob create
````

## /ob info
Kegunaan: Melihat info island<br>
Penggunaan: `/ob info [player]`<br>
Contoh dibawah:
```kominfo
/ob info
````

## /ob settings
Kegunaan: Pengaturan island<br>
Penggunaan: `/ob settings`<br>
Contoh dibawah:
```kominfo
/ob settings
````

## /ob setname
Kegunaan: Mengubah nama island<br>
Penggunaan: `/ob setname <nama>`<br>
Contoh dibawa:
```kominfo
/ob setname Island Dusty
```

## /ob resetname
Kegunaan: Reset nama island<br>
Penggunaan: `/ob resetname`<br>
Contoh dibawah:
```kominfo
/ob resetname
```

## /ob ban
Kegunaan: Ban orang dari island mu<br>
Penggunaan: `/ob ban <player>`<br>
Contoh dibawah:
```kominfo
/ob ban vrog
```

## /ob unban
Kegunaan: Unban orang dari island mu<br>
Penggunaan: `/ob unban <player>`<br>
Contoh dibawah:
```kominfo
/ob unban vrog
```

## /ob banlist
Kegunaan: Meliat list player yang kamu ban dari island<br>

Contoh dibawah:
```kominfo
/ob banlist
```
