# Crate
Fitur crate adalah sistem gacha di server.<br>
Hadiahnya bervariasi tergantung jenis crate.<br>
Untuk membuka crate, kalian butuh key yang sesuai.<br>
Ada 5 jenis crate: `Vote`, `Common`, `Rare`, `Builder`, dan `Bansos`<br>

## Vote Crate
Kalian butuh `Vote Key` untuk membuka **Vote Crate**.<br>
Lakukan [vote disini](https://turu.link/vote) untuk mendapatkan 4 key.
Buka crate ini di `/spawn`

## Builder Crate
Kalian butuh `Builder Key` untuk membuka **Builder Crate**.<br>
Menangkan giveaway untuk mendapat key ini.
Buka crate ini di `/warp crate`

## Bansos Crate
Kalian butuh `Kupon Bansos` untuk membuka **Bansos Crate**.<br>
Jawab kuis dari DustyGanteng untuk mendapatkan kupon.
Buka crate ini di `/warp bansos`
