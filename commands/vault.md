# Vault
Daftar command Vault yang dipakai untuk mengakses inventory tambahan.
Liat info tentang Vault [disini](/fitur/vault)<br>

## /pv
Kegunaan: Mengakses semua vault<br>
Contoh dibawah:
```kominfo
/pv
```

## /pv [nomor pv]
Kegunaan: Mengakses vault tertentu<br>
Contoh dibawah:
```kominfo
/pv 3
```
