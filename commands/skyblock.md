# SkyBlock
Daftar command SkyBlock bisa digunakan untuk buat island, invite orang, dan masih banyak lagi.<br>

## /is
Kegunaan: Teleport ke island atau membuat island jika belum ada<br>
Penggunaan: `/is`<br>
Contoh dibawah:
```kominfo
/is
```

## /is create
Kegunaan: Membuat island<br>
Penggunaan: `/is create`<br>
Contoh dibawah:
```kominfo
/is create
````

## /is info
Kegunaan: Melihat info island<br>
Penggunaan: `/is info [player]`<br>
Contoh dibawah:
```kominfo
/is info
````

## /is settings
Kegunaan: Pengaturan island<br>
Penggunaan: `/is settings`<br>
Contoh dibawah:
```kominfo
/is settings
````

## /is setname
Kegunaan: Mengubah nama island<br>
Penggunaan: `/is setname <nama>`<br>
Contoh dibawa:
```kominfo
/is setname Island punya Dusty
```

## /is resetname
Kegunaan: Reset nama island<br>
Penggunaan: `/is resetname`<br>
Contoh dibawah:
```kominfo
/is resetname
```

## /is ban
Kegunaan: Ban orang dari island mu<br>
Penggunaan: `/is ban <player>`<br>
Contoh dibawah:
```kominfo
/is ban vrog
```

## /is unban
Kegunaan: Unban orang dari island mu<br>
Penggunaan: `/is unban <player>`<br>
Contoh dibawah:
```kominfo
/is unban vrog
```

## /is banlist
Kegunaan: Meliat list player yang kamu ban dari island<br>
Penggunaan: `/is banlist`<br>
Contoh dibawah:
```kominfo
/is banlist
```

## /is team
Kegunaan: Pengaturan team island kamu<br>
Penggunaan: `/is team`<br>
Contoh dibawah:
```kominfo
/is team
```

## /is level
Kegunaan: Meliat level kamu atau player lain<br>
Penggunaan: `/is level [player]`<br>
Contoh dibawah:
```kominfo
/is level
```

## /is top
Kegunaan: Top island berdasarkan level island<br>
Penggunaan: `/is top`<br>
Contoh dibawah:
```kominfo
/is top
```

## /is warp
Kegunaan: Teleport ke island orang<br>
Penggunaan: `/is warp [player]`<br>
Contoh dibawah:
```kominfo
/is warp
```

## /is teamchat
Kegunaan: Mengubah chat dari global ke island. Kalau mau ke chat global ketik command nya lagi.<br>
Penggunaan: `/is teamchat`<br>
Contoh dibawah:
```kominfo
/is teamchat
```

## /is chat
Kegunaan: Mematikan atau menghidupkan chat di island<br>
Penggunaan: `/is chat`<br>
Contoh dibawah:
```kominfo
/is chat
```
