# Vault
Vault adalah fitur yang berguna untuk mengakses inventory tambahan.<br>
Ini bisa jadi alternatif dari enderchest dan shulker box.<br>
Totalnya ada 54 vault, setiap vault ada 27 slot item.<br>
Liat command nya [disini](/commands/vault)

## Permission
1. Rank Default: Vault #1 - #2
2. Rank VIP: Vault #1 - #7
3. Rank VIP+: Vault #1 - #12
4. Rank MVP: Vault #1 - #17
5. Rank MVP+: Vault #1 - #22
6. Rank MVP++: Vault #1 - #29
7. Rank Legend: Vault #1 - #39
8. Rank Supporter: Vault #1 - #54
